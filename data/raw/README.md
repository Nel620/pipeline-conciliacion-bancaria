# Datos de ejemplo

Esta carpeta contiene archivos de prueba diseñados para demostrar el funcionamiento del pipeline.

Los archivos incluyen distintos escenarios, como:

- Valores nulos.
- Registros duplicados.
- Fechas en diferentes formatos.
- Inconsistencias en nombres de empresas.
- Errores comunes de calidad de datos.

Estos archivos son únicamente para fines demostrativos y no contienen información real.
