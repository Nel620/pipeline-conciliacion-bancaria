# Datos Procesados

Esta carpeta contiene los archivos generados por el pipeline de conciliación bancaria después de ejecutar las etapas de limpieza, validación y transformación de los datos.

> **Importante:** El archivo incluido en esta carpeta corresponde al resultado del procesamiento del archivo `05_reporte_mixto_empresas_2026_01.xlsx`, ubicado en `data/raw/`. Los demás archivos de la carpeta `raw/` se incluyen como conjuntos de datos de prueba para que cualquier persona pueda ejecutar y probar el proyecto con diferentes escenarios.

Los reportes, gráficas y demás resultados generados por el pipeline se encuentran en la carpeta `data/output/`.
